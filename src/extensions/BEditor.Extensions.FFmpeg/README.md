# BEditor.Extensions.FFmpeg

## Description
FFmpegを使って動画のデコード、エンコード機能を提供するライブラリ

## Dependent libraries
* [.NET Runtime](https://github.com/dotnet/runtime)
* [FFMediaToolkit](https://github.com/radek-k/FFMediaToolkit)
* [FFmpeg](https://ffmpeg.org)
